# Next.js OAuth Learning App

A simple authentication app built with **Next.js**, designed to help understand and implement **OAuth** using popular providers like Google, GitHub, and more. This project focuses on real-world OAuth flows, session management, and secure access handling.

## 🚀 Features

- 🔐 OAuth login (Google, GitHub, etc.) via `next-auth`
- 📦 Session handling with JWT and cookies
- 🧑 Protected routes and user dashboard
- 💡 Clean and minimal UI for learning purposes
- 🌐 Server-side rendering support

## 🛠️ Technologies Used

- [Next.js](https://nextjs.org/)
- [NextAuth.js](https://next-auth.js.org/)
- [Tailwind CSS](https://tailwindcss.com/) (optional)
- TypeScript (optional, if you’re using it)
